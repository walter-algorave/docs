## Cos'è PokerTH Companion?

**PokerTH Companion** è un'applicazione Windows che osserva passivamente una o più finestre di PokerTH (simulatore Texas Hold'em open-source) tramite **Computer Vision**, estrae lo stato di gioco in tempo reale senza iniettare codice e fornisce suggerimenti decisionali preflop basati su tabelle GTO.

Il sistema è progettato come caso di studio accademico sull'applicazione di tecniche CV all'estrazione dati da applicazioni desktop a scatola chiusa.

<Note>
Il programma è un **Strumento di Supporto Decisionale passivo** — non interagisce mai con PokerTH, non simula input e non viola nessuna policy. PokerTH è open-source e non usa denaro reale.
</Note>

## Architettura a colpo d'occhio

```
Presentazione  →  Engine (applicazione)  →  Vision (dominio)  →  Infrastruttura
   gui.py            session.py              card_reader.py       screen_capture.py
   gui_views.py       hand_history.py         table_reader.py      settings.py
   calibration.py     decision_engine.py      image_utils.py       templates/cards/
```

Il ciclo di analisi gira a **1 Hz** in un thread daemon: cattura → estrae stato → confronta → notifica via Observer → GUI aggiorna.

## Stack tecnologico

| Libreria | Versione | Ruolo |
|---|---|---|
| `opencv-python` | ≥ 4.8 | Template matching, HSV, preprocessing immagini |
| `pytesseract` / Tesseract | ≥ 0.3.10 | OCR per valori numerici (bet, pot) |
| `pywin32` | ≥ 306 | `PrintWindow`, `EnumWindows`, cattura background |
| `customtkinter` | ≥ 5.2.2 | GUI moderna in dark mode |
| `Pillow` | ≥ 9.0 | Conversione buffer Win32 → numpy array |
| `numpy` | ≥ 1.24 | Manipolazione array immagine |

## Limiti attuali

<Warning>
- **Windows only** — dipende da Win32 API (`PrintWindow`, `GetWindowDC`)
- **Risoluzione fissa** — 1280×720 px (il programma ridimensiona PokerTH automaticamente)
- **Solo preflop** — il DSS non copre flop/turn/river
- **No 3-bet / limp** — la logica VS_RAISE assume un singolo raiser
</Warning>
