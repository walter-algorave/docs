## `seat_to_position(seat, dealer, active_players) â†’ str`

Calcola la posizione relativa dell'hero dato il seat dealer e i giocatori attivi.

```python
def seat_to_position(seat: int, dealer: int, active_players: list = None) -> str:
    if dealer < 0 or not active_players or seat not in active_players:
        return "Unknown"

    # Costruisce lista ordinata in senso orario da dealer
    action_order = [(dealer + i) % 6 for i in range(6)]
    ordered_active = [s for s in action_order if s in active_players]

    player_index = ordered_active.index(seat)
    return _POS_MAP.get(player_index, "Unknown")
```

**Comportamento con tavoli short-handed**: con meno di 6 giocatori, le posizioni si "comprimono" automaticamente â€” sono presenti solo le posizioni che corrispondono ai giocatori attivi, mantenendo BU/SB/BB come ultime tre.

### Esempi

| Dealer | Hero Seat | Attivi | Posizione Hero |
|---|---|---|---|
| 0 | 0 | tutti | BU |
| 1 | 0 | tutti | CO |
| 2 | 0 | tutti | MP |
| 5 | 0 | tutti | SB |
| 4 | 0 | [0, 4] | BB (heads-up) |

### Valori di ritorno

- `"BU"` / `"SB"` / `"BB"` / `"EP"` / `"MP"` / `"CO"` â€” posizione calcolata
- `"Unknown"` â€” dealer non rilevato (`dealer < 0`), lista attivi vuota, o seat non nella lista attivi
